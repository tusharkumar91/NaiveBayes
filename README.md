# NaiveBayes

Algorithms Implemented :
* Naive Bayes 
* Gaussian Naive Bayes classifier 

Dataset used:
* Sms spam dataset for Naive Bayes 
* Pima Indians Diabetes dataset for Gaussian(https://www.kaggle.com/uciml/pima-indians-diabetes-database)
